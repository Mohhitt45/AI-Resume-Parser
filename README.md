# AI Resume Parser (NLP + FastAPI)

## 🚀 Overview
AI-powered system that extracts structured information from resumes using NLP techniques.

## ⚙️ Features
- PDF resume parsing
- Name, email, phone extraction
- Skill detection
- JSON structured output
- REST API using FastAPI

## 🧠 Architecture
PDF → Text Extraction → NLP Parsing → JSON Output → API Response

## ▶️ Run Project
uvicorn app:app --reload

## 📌 API
POST /parse_resume
